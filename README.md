# kol-desc
A collection of miscellaneous kolmafia relay scripts for Kingdom of Loathing.

## Installation:
Run the following in the Graphical CLI in KolMafia:

```git checkout deadned24/kol-desc```

## Description:
KoLmafia relay script for the web game Kingdom of Loathing (kol). Enhances in‑game pop‑up descriptions with extra data for items, effects, familiars, outfits, and skills. It displays food category, adventure gain, item ID, and “Last Available Date” (Standard availabilty date), and adds direct links to the wiki and mall.
