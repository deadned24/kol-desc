# kol-desc
A collection of kolmafia relay scripts for Kingdom of Loathing to enhance popup descriptions. Installing will add relay overrides to the following pages:
- desc_effect.php
- desc_familiar.php
- desc_item.php
- desc_outfit.php
- desc_skill.php

## Installation:
Run the following in the Graphical CLI in KolMafia:

```git checkout deadned24/kol-desc```

## Description:
KoLmafia relay script for the web game Kingdom of Loathing (kol). Enhances in‑game pop‑up descriptions with extra data for items, effects, familiars, outfits, and skills. It displays food category, adventure gain, item ID, and “Last Available Date” (Standard availabilty date), adds direct links to the wiki and mall, and more.
